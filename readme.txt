== Description ==

Tweets for wordpress fetches and displays public tweets of a user.
get_tweets(<username>, <number_of_tweets>, <is_linked_to_wordpress>, <links_clicked_in_a_new_browser_window>);
**Features**

    *  Simple
    *  Customizable
    *  Uses Wordpress resources (no extra files needed)

**Usage**


`<?php get_tweets("username", 10, true, true); ?>`

For more info (options, customisation, etc.) visit [the plugin homepage](http://ramsich.com/twitter-in-wordpress-plugin/#.UQucw0JZHRU "Tweets for Wordpress").

**Customization**

The plug in provides the following CSS classes:

    * ul.twitter: the main ul (if list is activated)
    
== Installation ==

Drop tweets_in_Wordpress folder (or even twitter.php) into /wp-content/plugins/ and activate the plug in the Wordpress admin area.
